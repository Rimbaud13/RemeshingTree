All the necessary information for this project can be found in the `report.pdf`
file.

The `Code` folder contains all the code which procudes our final mesh. 
Instructions to run it can be found in the "Instructions" chapter of the 
`report.pdf` file.

`final_mesh.obj` contains the final mesh of our project, which is ready to 
print.

The `video.avi` file is a video which briefly shows the GUI for our project
as well as a short simulation of the mesh used as a lamp.
